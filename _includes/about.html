What is mbtBundle?

MBT Bundle is an open-source tool that allow you to test your application using Model Based Testing technique. Everytime a new version of your application released, this tool will test all business logic to make sure all features are working correctly. Every bug will be reported automatically e.g. to Slack or mail etc. QA do not need to write any test cases, or maintain or execute any automation test cases. Instead, QA will focus on business logic e.g. to make sure business logic do not have any defects, or all business logic are implemented.

Open Source

It's open source, so it's always free.

Time saver

QA does not need to write, execute or maintain any test cases. It automatically make reproduce steps as shortest as possible. Bugs are automatically reported.

Flexible

Can either do exploration test your application to find more bugs, or can test it in a limit timebox. It can be used to test API application, or Web application.

Agile-friendly

Allow to test your application even though it's not implemented. Allow team member focus on business logic.

Scalable

Built with scalable in mind. Can test an application with a lot of business logic.

User-friendly

It has UI and API. It use a friendly file format (yaml) to define business logic. Developer write code to interact with system-under-test directly without any restriction.
